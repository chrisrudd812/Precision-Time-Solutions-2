Mobile App Testing Checklist

1. Prerequisites

  1. Phone and computer on same WiFi network
  2. Expo Go app installed on phone
  3. Eclipse running with your Java web app at localhost:8082

2. Starting Development Server

  1. Open PowerShell/Terminal
  2. Navigate to project: cd C:\MobileApps\TimeclockApp
  3. Start Expo: npx expo start
  4. Wait for QR code to appear in terminal

3. Connect Phone to Development Server
  1. Open Expo Go app on phone
  2. Option A: Scan QR code from terminal
  3. Option B: Manual entry - look for URL like exp://
  4. Wait for QR code to appear in terminal
6. When Making Code Changes
  1. Save file in VS Code
  2. Press r in terminal to reload app
  3. Check phone for updates
7. If Connection Lost
  1. Press Ctrl+C in terminal to stop server
  2. Run npx expo start again
  3. Reconnect phone using QR code or manual URL
8. Troubleshooting
  1. If 404 errors: Check Eclipse server is running
  2. If network errors: Verify both devices on same WiFi
  3. If app won't update: Try r to reload, or restart Expo server
Order: Eclipse server first, then Expo server, then connect phone.
